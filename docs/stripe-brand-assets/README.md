# SendaFit - Assets para Stripe

Paquete de marca preparado para configurar la cuenta de Stripe de SendaFit.

## Archivos

- `sendafit-stripe-icon-512.png`: icono principal recomendado para Stripe.
- `sendafit-stripe-icon-256.png`: variante ligera si Stripe pide un archivo mas pequeno.
- `sendafit-stripe-icon-128.png`: variante compacta para usos secundarios.
- `sendafit-stripe-logo-horizontal-transparent.png`: logo horizontal con fondo transparente.
- `sendafit-stripe-logo-horizontal-white.png`: logo horizontal sobre fondo blanco, recomendado si Stripe muestra baja legibilidad con transparencias.
- `sendafit-stripe-banner-1600x500.png`: banner de marca para perfiles, portales o materiales donde Stripe permita imagen amplia.

## Recomendacion de uso en Stripe

- Icono de la empresa/app: usar `sendafit-stripe-icon-512.png`.
- Logo de marca: usar `sendafit-stripe-logo-horizontal-white.png` si Stripe lo muestra sobre fondo claro; usar `sendafit-stripe-logo-horizontal-transparent.png` solo si Stripe conserva buen contraste.
- Color de marca: `#a855f7`.
- Color secundario/acento: `#ec4899`.
- Nombre comercial: `SendaFit`.
- Nombre del producto actual: `SendaFit Pro`.

Estos assets son derivados del icono propio de SendaFit en `public/sendafit-icon.svg` y no incluyen assets de Lovable.
