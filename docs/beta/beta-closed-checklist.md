## Flujo base

- [ ] Registro completado.
- [ ] Onboarding completado.
- [ ] Perfil guardado correctamente.
- [ ] Rutina asignada.
- [ ] Pantalla Hoy muestra entrenamiento correcto.
- [ ] Entrenar día 1.
- [ ] Completar entrenamiento.
- [ ] Ver entrenamiento en historial/progreso.
- [ ] Mover entrenamiento.
- [ ] Saltar entrenamiento.
- [ ] Editar perfil y confirmar que no se pierden datos.

## Nutrición

- [ ] Registrar comida manual.
- [ ] Analizar comida con IA.
- [ ] Corregir macros antes de guardar.
- [ ] Confirmar que las calorías por foto aparecen como estimación.

## Entrenamiento e IA

- [ ] Sustituir ejercicio.
- [ ] Analizar máquina con IA.
- [ ] Guardar ejercicio de máquina como favorito o aplicarlo.
- [ ] Hablar con coach.
- [ ] Preguntar por dolor/lesión y confirmar respuesta segura.
- [ ] Aplicar una rutina sugerida solo después de confirmación.

## PWA y offline

- [ ] Instalar en Android o iOS.
- [ ] Abrir desde pantalla de inicio.
- [ ] Navegar sin conexión.
- [ ] Confirmar banner offline.
- [ ] Volver online sin perder datos recientes.

## Feedback

- [ ] Enviar reporte de error.
- [ ] Enviar sugerencia.
- [ ] Enviar reporte de confusión.
- [ ] Enviar pantalla problemática.

## Cierre

- [ ] No hubo pérdida de datos.
- [ ] Se registraron al menos 5 mejoras reales.
- [ ] Tester usaría la app 7 días más.
- [ ] Tester entiende límites de IA/salud.
