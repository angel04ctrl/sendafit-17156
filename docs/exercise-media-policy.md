# Politica de multimedia para ejercicios

Sprint 6.1 y 6.1.2 no agregan imagenes ni videos externos a la biblioteca de ejercicios.

Reglas:

- No usar imagenes o videos sin licencia clara.
- No copiar contenido de redes sociales, YouTube, blogs o bancos de imagenes sin permiso explicito.
- Mantener `imagen` y `video` en `null` cuando no exista una fuente confiable.
- Las instrucciones paso a paso, explicacion tecnica, cues, errores comunes y contraindicaciones son el fallback principal cuando no hay multimedia.
- Para una futura expansion se debe evaluar una fuente abierta con licencia compatible y guardar la atribucion correspondiente.

Riesgo pendiente:

La experiencia visual del detalle de ejercicio depende principalmente del texto tecnico hasta que exista una fuente multimedia segura.
